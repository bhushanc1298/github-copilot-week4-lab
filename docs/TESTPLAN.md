## Test Execution Notes

- All tests assume the application starts with $1000.00 initial balance
- Balance displays use COBOL PIC 9(6)V99 format (leading zeros)
- Tests should be executed in sequence for cumulative operations
- For persistence test (TC011), note that current COBOL implementation may not maintain state between program executions
- Invalid input tests (negative amounts) may need additional validation logic in future Node.js implementation
